# Tools

Shell scripts and utilities that support PradOS operations.

## Adding a Tool

Place the script here with a descriptive name.
Add an entry to this README with:
- What it does
- How to run it
- What it produces

Simple tools: a single `.sh` or `.py` file.
Complex tools: a folder with `run.py`, `README.md`, and optionally `config.json`.
