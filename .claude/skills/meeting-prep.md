---
name: meeting-prep
description: Pull context for an upcoming meeting from Knowledge/ and recent work
---

Ask for the meeting topic or person's name if not provided.

Then search across:
1. Knowledge/People/ for a file matching the person's name
2. Meetings/1on1s/ for recent notes with that person
3. Projects/ for any active projects involving them or the topic
4. Tasks/active.md for any tasks related to them or the topic

Produce a prep sheet:

**Meeting:** [Topic / Person]
**Date:** [If known]

**Context:**
[Key facts from Knowledge/People/ or prior meeting notes]

**Open items from last time:**
[Action items that were assigned, if found in prior notes]

**Suggested agenda:**
1. [Item based on open actions]
2. [Item based on active project status]
3. [Any blockers or decisions needed]

**Notes to remember:**
[Any important preferences or context from Knowledge/People/]

Ask: "Should I save this as a meeting prep note in Meetings/1on1s/ or _temp/?"
