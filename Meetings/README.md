# Meetings

Meeting notes organized by type.

## Structure

```
Meetings/
├── 1on1s/           ← Recurring 1on1 notes: YYYY-MM-DD-[person-name].md
└── standups/        ← Daily standup notes: YYYY-MM-DD.md
```

## File Naming

- 1on1s: `YYYY-MM-DD-[person-name].md`
- Standups: `YYYY-MM-DD.md`

## Templates

Use `Templates/meeting-notes.md` or `Templates/1on1.md` as starting point.
