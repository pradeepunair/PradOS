# _temp — Dump Area

No organization required here. Capture first, organize later.

## Rules

- Anything goes in here
- Files older than 30 days should be reviewed and either moved or deleted
- If something in _temp becomes useful, promote it to the right folder
- Claude will suggest promotions when it notices patterns

## Filename convention (loose)

`YYYY-MM-DD-[topic].md` works well but is not required.
